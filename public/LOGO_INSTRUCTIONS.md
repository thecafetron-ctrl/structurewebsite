# Logo Setup

Save your logo image as `logo.png` in this `/public` folder.

The logo you uploaded should be placed here with the filename: **logo.png**

It will automatically appear in the header.

## Image Specifications

- Format: PNG (transparent background recommended)
- Dimensions: Minimum 100x100px
- File name: `logo.png`
- Location: `/public/logo.png`

The site is already configured to use `/logo.png` in the header component.
